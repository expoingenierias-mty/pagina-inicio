Urban Vlogger: Towards a Conscious Mobility

The purpose of the project is to obtain relevant, real, and useful information about the mobility of a city, to make the population aware of the C02 emissions that are produced when traveling from one place to another. Likewise, the results help decision-makers to plan and design changes in the urban structure to improve mobility and create a sustainable and smart city. This is achieved through automated video-analysis using object detection and tracking algorithms in a specific path or space in urban environments.
