Control Bilateral para Interfaz Háptica

Diseño de un prototipo de garra (gripper) en la que el usuario pueda tener un control bilateral; es decir, controlar a distancia con su propia mano la garra. El objetivo es lograr que al momento que el usuario cierra su mano, el prototipo debe replicar esa misma acción cerrándose. Si este no puede continuar la acción (por ejemplo, al encontrarse con un obstáculo), envía una señal para que la pinza controlada por el usuario reciba un torque contrario; previniendo al usuario de seguir cerrando su mano. Con esto, se busca codificar información sobre el ambiente de la pinza teleoperada como señales mecánicas que el usuario pueda sentir, siendo esta la base de un “tacto” artificial.
