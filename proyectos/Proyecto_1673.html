Real-time Biofeedback System for Interactive Learning using Wearables and IoT

Via biometry, using multiple devices interconnected using an IoT architecture, to develop a cloud-based infrastructure with AI-based algorithms to predict relevant cognitive features in students while taking lectures, in order to improve their performance and thus education.
