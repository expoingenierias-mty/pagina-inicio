DESARROLLO DE PROTOTIPO DE SOFTWARE

Adaptación a una plataforma de teléfono celular del algoritmo de aprendizaje profundo para la detección de enfermedades torácicas

Se tomará el modelo ganador de la competencia de CheXpert y se exportara a una version movil para despues construir una aplicacion movil (android). Esto constituye una manera de hacer accesible una herramienta tecnologíca de alto nivel a comunidades que no cuenten con expertos médicos para el diagnóstico de enfermedades.
