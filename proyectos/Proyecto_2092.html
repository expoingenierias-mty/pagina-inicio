DESARROLLO DE PROTOTIPO FISICO

Sistema predictivo para la reducción de tiempos de paros en línea de producción

El nombre del proyecto a realizar recibió el nombre de “Sistema predictivo para la prevención de paros prolongados”, este reto nace de la necesidad de reducir el tiempo muerto y el tiempo de reacción al abordar una falla en la línea de producción, ya que actualmente el socio formador no cuenta con un sistema el cual sea capaz de reducir estos tiempos. Estos tiempos son de suma importancia para el socio formador, debido a que nos dictan cuanto es el tiempo que se tiene que parar la línea mientras se está solucionando una falla técnica y en muchas de las ocasiones como no se cuenta con un sistema para la prevención de fallas, lo que ocasiona que el paro técnico sea demasiado prolongado, significando en una gran pérdida de unidades completadas; un minuto de paro pude significar una gran pérdida para el socio formador.
