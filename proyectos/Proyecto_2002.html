(SIMAGUA AV) Vehículo Autónomo para el Monitoreo de Cuerpos de Agua Dulce

SIMAGUA AV se trata de un bote autónomo de dos propelas capaz de realizar recorridos alrededor de lagunas, rios, estanque de peces, granjas de acuicultura y sensar distintos parametrós del agua para posteriormente enviar estos datos en tiempo real a una pagina web donde puedan ser visualizados de manera gráfica, las trayectorias pueden ser programadas y modificadas por medio de una aplicación con coordenadas gps, de manera sencilla, no obstante es posible utilizar en su lugar un control remoto para realizar los recorridos. Nuestro proyecto es además autónomo energéticamente ya que tiene integrado paneles solares para la carga de energía, utiliza supercapacitores como medio de almacenaje eléctrico lo cual lo hace amigable con el medio ambiente.
